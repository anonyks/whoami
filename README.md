# whoami

my personal site. terminal-themed portfolio thing.

built with react + vite + tailwind. has a fake boot sequence, a terminal info section, pulls my github repos live, and some other stuff.

### run it

```
bun install
bun dev
```

or npm, whatever:

```
npm install
npm run dev
```

opens on `localhost:8080`

### build

```
bun run build
```

### what's in here

- boot sequence animation on load
- terminal-style intro (`$ whoami`)
- about, interests, featured work, projects (fetched from github api)
- interactive terminal input (try typing stuff)
- ask anonyks chatbot thing
- contact section

### stack

react, typescript, vite, tailwind, framer-motion, shadcn/ui bits, radix
